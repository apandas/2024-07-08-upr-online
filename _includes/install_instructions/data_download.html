Step 6. Data Files

In order to follow along with the workshop on your device, you will need to download the data we will be using.

This is a list of data files you will need to download and move to your desktop.

    1. Data for the Unix Shell lesson. Download shell-lesson-data.zip. Unzip/extract the file. You should end up with a new folder called shell-lesson-data on your Desktop.
    2. Data for the Python lesson. The data we will be using is taken from the gapminder dataset. To obtain it, download and unzip the file python-novice-gapminder-data.zip. After extracting and moving the files onto your desktop, you should have a folder named "data" on your desktop
